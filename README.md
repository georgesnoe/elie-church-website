# elie-church-website
Un site web pour le site web de l'église d'un ami

Ce site web a été conçu avec PHP et MySql

## Composition du projet
Ce projet est composée d'une interface utilisateur par défaut,
sur laquelle les utilisateurs pourront accéder aux informations
qu'ils désirent.

Il y'a également une interface admin sur laquelle l'administrateur
pourra modifier à la volée, les informations qu'il souhaite
afficher à l'utilisateur.
